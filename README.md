## Installation
```bash
git clone --depth=1 https://github.com/spql-lang/deb.git spql
cd spql
chmod +x install.sh
sudo ./install.sh
```

## Uninstallation
```bash
chmod +x uninstall.sh
sudo ./uninstall.sh
```
